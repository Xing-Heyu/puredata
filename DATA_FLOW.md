# PureData 数据生成流程图

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           PureData 数据生成完整流程                               │
└─────────────────────────────────────────────────────────────────────────────────┘

用户请求 (domain, count, quality_mode)
    │
    ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│  simple_main.py: generate_data_clean()                                          │
│  入口函数 - 根据质量模式配置流水线                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│  PipelineConfig (质量模式配置)                                                   │
│                                                                                  │
│  ┌──────────────┬──────────────┬──────────────┬──────────────┐                  │
│  │   standard   │    high      │   ultra      │   mixed      │                  │
│  │   50%高质量  │   80%高质量  │   95%高质量  │   自定义      │                  │
│  └──────────────┴──────────────┴──────────────┴──────────────┘                  │
│                                                                                  │
│  + enable_modules: ["cads", "diversity", ...]  ← 自定义模块启用                  │
│  + auto_enhance: True                          ← 自动质量提升                    │
└─────────────────────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│  HighQualityGenerator (数据生成)                                                 │
│  ├─ 调用 千问API集成.py (QwenAPI)                                               │
│  └─ 三层知识来源：知识库 → API → 模板兜底                                         │
└─────────────────────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           DataQualityPipeline                                    │
│                           数据质量流水线                                          │
└─────────────────────────────────────────────────────────────────────────────────┘
    │
    ├──[阶段1]──▶ T²质量控制 (t2_quality_control.py)
    │             ├─ Team: 多生成器协作
    │             └─ Trim: 质量修剪
    │
    ├──[阶段2]──▶ 专业验证 (professional_validator.py)
    │             ├─ 领域术语验证
    │             ├─ 专业约束检查
    │             └─ 错误检测与修正建议
    │
    ├──[阶段3]──▶ 异常检测 (anomaly_detector.py)
    │             ├─ 基于国家标准的异常检测
    │             └─ 自动修复 (auto_fixer)
    │
    ├──[阶段4]──▶ 去重系统 (deduplication_system.py)
    │             ├─ MinHash LSH 近似去重
    │             └─ 精确去重 (MD5)
    │
    ├──[阶段5]──▶ 多样性增强 (diversity_enhancer.py) [high/ultra模式]
    │             ├─ GECE长尾检测 (ACL2024)
    │             ├─ 分布外增强
    │             └─ 对抗式生成
    │
    ├──[阶段6]──▶ 质量门控 (quality_gate.py)
    │             ├─ 四级质量分类
    │             │   ├─ high_quality (≥0.85)
    │             │   ├─ free_quality (0.80-0.85)
    │             │   ├─ medium_quality (0.75-0.80)
    │             │   └─ robustness_quality (<0.75)
    │             └─ 自动重试机制
    │
    ├──[阶段7]──▶ LLM审计 (llm_data_auditor.py) [high/ultra模式]
    │             └─ 9维度评估 (arXiv:2601.17717)
    │
    ├──[学术模块]──▶ 根据质量模式启用 [ultra模式全部启用]
    │
    │   ├─ CADS对抗合成 (CADS对抗合成.py)
    │   │   └─ AI互相挑错提升质量
    │   │
    │   ├─ DASGen分布对齐 (DASGen分布对齐生成.py)
    │   │   └─ 长尾语义增强
    │   │
    │   ├─ 真实种子数据 (真实种子数据.py)
    │   │   └─ 从真实演进中学习
    │   │
    │   ├─ 增强数据生成器 (增强数据生成器.py)
    │   │   └─ 情绪建模、意外事件
    │   │
    │   ├─ 本地知识图谱 (本地知识图谱生成.py)
    │   │   └─ 离线任务合成
    │   │
    │   ├─ FAC特征覆盖 (FAC特征覆盖合成.py)
    │   │   └─ 特征激活覆盖度
    │   │
    │   └─ 失败数据回收 (失败数据回收.py) [high/ultra模式]
    │       └─ 子任务分解重试
    │
    └──[智能增强]──▶ 自动质量提升 [auto_enhance=True时]
                    └─ 检测质量不足时自动启用额外模块
    │
    ▼
┌─────────────────────────────────────────────────────────────────────────────────┐
│                           输出数据结构                                           │
└─────────────────────────────────────────────────────────────────────────────────┘
{
    "id": 1,
    "word": "深度学习",
    "text": "深度学习是机器学习的子领域...",
    "category": "人工智能",
    "source": "knowledge_base | qianwen_api | template_fallback",
    "quality_score": 0.85,
    "quality_level": "high_quality",
    "pipeline_quality_score": 0.88,
    "pipeline_quality_level": "high_quality",
    "provenance": {
        "platform": "PureData",
        "version": "3.0.0",
        "quality_mode": "high",
        "generated_at": "2026-02-22T...",
        "stages_passed": ["t2_control", "professional_validation", ...]
    }
}
```
