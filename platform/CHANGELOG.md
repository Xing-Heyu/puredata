# 更新日志

## [1.0.0] - 2026-07-05

### 开源首发
- 多领域数据生成（人工智能、医疗、金融、劳动合同、交通驾驶）
- 多输出类型（文本、知识图谱、事件链、文献、多模态）
- 统一质量管道（T²质量控制、专业验证、异常检测、去重、多样性增强）
- 7 阶段质量处理流程
- 学术模块（CADS、DASGen、FAC 等）

### 核心功能
- 质量门控（四级分类）
- LLM 审计（9 维度评估）
- 专业验证（领域术语检查）
- 去重系统（MinHash LSH）
- 多样性检测（长尾覆盖）
- 异常检测（基于国家标准）
- MinHash LSH 近似去重
- 多格式导出（JSON/JSONL/CSV/Parquet）
