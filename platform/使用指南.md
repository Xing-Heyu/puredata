## 🚀 快速开始

### 第一步：打开终端

按 `Win + R`，输入 `cmd` 或 `powershell`，回车打开终端。

### 第二步：进入项目目录

在终端输入以下命令：

```bash
cd d:\skill\skill3\platform\backend
```

在终端输入以下命令：


## 🎯 功能说明

### 1. 数据生成

支持三种生成模式：

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **Hybrid** | 混合模式，30%干净+50%噪声+20%结构化 | 推荐，通用场景 |
| **Clean** | 干净数据，无噪声 | 预训练、精调 |
| **Noisy** | 噪声数据，模拟真实数据 | 鲁棒性训练 |

### 2. 支持领域

- 人工智能
- 劳动合同
- 医疗
- 金融
- 支持定制领域

### 3. 输出格式

| 格式 | 说明 |
|------|------|
| JSON | 通用格式，易解析 |
| JSONL | 行式JSON，适合流处理 |
| CSV | 表格格式，Excel兼容 |
| TXT | 纯文本格式 |
| Parquet | 企业版专属，大数据优化 |

### 4. 数据质量

系统自动进行：
- ✅ 敏感信息脱敏（手机号、身份证、银行卡等）
- ✅ 不良内容过滤
- ✅ 质量评分报告
- ✅ 优化建议

## 🔌 API使用

### 认证方式

```bash
# 登录获取Token
curl -X POST http://localhost:8000/api/login -H "Content-Type: application/json" -d "{\"username\":\"your_username\",\"password\":\"your_password\"}"
```

### 生成数据

```bash
curl -X POST http://localhost:8000/generate -H "Authorization: Bearer YOUR_TOKEN" -H "Content-Type: application/json" -d "{\"domain\":\"人工智能\",\"count\":100,\"format\":\"json\",\"mode\":\"hybrid\"}"
```

### 查询任务状态

```bash
curl http://localhost:8000/task/TASK_ID
```

## ❓ 常见问题

### Q: 启动报错"python不是内部命令"？

A: 说明Python未安装或未添加到环境变量。
1. 下载安装Python：https://www.python.org/downloads/
2. 安装时勾选"Add Python to PATH"
3. 重启终端后再试

### Q: 启动报错"ModuleNotFoundError"？

A: 缺少依赖，在终端输入：
```bash
cd d:\skill\skill3\platform\backend
pip install -r requirements.txt
```

### Q: 端口8000被占用？

A: 修改端口：
1. 打开 `backend/simple_main.py`
2. 找到最后一行 `server.serve_forever()`
3. 将端口改为其他值，如8080

### Q: 忘记管理员密码？

A: 在终端输入以下命令重置：
```bash
cd d:\skill\skill3\platform\backend
python -c "open('管理员配置.json','w',encoding='utf-8').write('{\"admins\": [{\"username\": \"admin\", \"password_hash\": \"240be518fabd2724ddb6f04eeb1da5967448d7e831c08c8fa822809f74c720a9\", \"email\": \"admin@datagenpro.com\"}], \"settings\": {\"session_timeout_hours\": 24}}')"
```
密码将重置为：`admin123`

### Q: 如何获取更多免费额度？

A: 
1. 邀请好友注册，双方各得500条
2. 完成任务：生成满1000条送500条
3. 升级付费套餐
